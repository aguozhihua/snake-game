# 🐍 Snake Game (HTML5)

[![Play Online](https://img.shields.io/badge/Play-Online-success?style=for-the-badge&logo=github)](https://aguozhihua.github.io/snake-game/)

A classic Snake game implementation with a **Cyberpunk AI** aesthetic.

## 🎮 Play Now

👉 **[Click here to play!](https://aguozhihua.github.io/snake-game/)**

## ✨ Features

- **Cyberpunk Visuals**: Neon glow, particle explosions, and dynamic grid.
- **Arrow Keys** (⬆️ ⬇️ ⬅️ ➡️) to control the snake.

### Mobile
- **Swipe** on the game board to change direction.
- Tap the **on-screen buttons** if you prefer touch controls.

## ✨ Features

- **Responsive Design**: Works on desktop and mobile.
- **Touch Controls**: Optimized swipe detection for mobile devices.
- **Score Tracking**: Real-time score updates.
- **Single File**: All code (HTML, CSS, JS) is contained in `index.html` for easy sharing.

## 🚀 Quick Start

1. **Download** the code:
   ```bash
   git clone https://github.com/aguozhihua/snake-game.git
   ```
2. **Open** `index.html` in any modern web browser.
3. **Enjoy!**

## 🛠️ Customization

You can tweak the game settings in the `<script>` section of `index.html`:

```javascript
const BOARD_SIZE = 20;  // Grid size (20x20)
const PIXEL_SIZE = 20;  // Size of each block in pixels
const INITIAL_SPEED = 150; // Game speed (lower is faster)
```
